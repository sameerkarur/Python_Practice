# Python Practice - Data Analysis & Data Science

A repository for practicing Python coding focused on data analysis and data science.

## Folder Structure

- **numpy/** - NumPy practice (arrays, linear algebra, numerical computing)
- **pandas/** - Pandas practice (DataFrames, data manipulation, cleaning)
- **matplotlib/** - Matplotlib practice (data visualization, plotting)
- **seaborn/** - Seaborn practice (statistical data visualization)
